## Brand Structure

### Series Model

Products are organized into **series**:

| Series Type | Description |
|-------------|-------------|
| **Ongoing** | Continuous releases (e.g., Bitcoin Series) |
| **Limited Edition** | Time-bound or quantity-limited drops |

### Entry Mechanism

Badge-gated minting with two-tier system:

| Tier | Badge Type | Purpose |
|------|------------|---------|
| 1 | Entry Badge | Gates initial Vault access |
| 2 | Achievement | Enables Vault stacking |

### Governance

Sole issuer (personal brand model):
- Decision-making: Personal authority
- Accountability: Brand reputation
- Speed: Fast iteration
- Trust: Brand-dependent
